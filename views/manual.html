소독리포트 요약

OASIS

소독기 사용자 메뉴얼

- 회원가입

  • ① 구글 플레이스토어 [오아시스 관리자] 검색 후 설치 진행
  • ② 개인 구글 계정으로 회원가입 진행
  • ③ 사업자 정보에 상호 입력 후 가입 승인 요청 진행
  • ④ 관리자 WEB서버(www.cleanoasis.net) 접속 후 작업자 승인 요청 처리 진행
  • ⑤ [사업자 권한] 부여 시 관리자 어플 내 [모든 기능] 사용가능
  • ⑥ [승인] 시 [차량소독, 소독이력, 마이페이지]에 한정된 기능 사용가능

- 차량소독

  • ① 차량 소독메뉴 클릭 시 연결 가능한 소독기 자동으로 확인 가능 (B/T 연결 필수)
  • ② 장비 선택 시 ③으로 이동
  • ③ 차량번호 입력 (99가9999 입력)
  • ④ 희망 작업 코스 선택
    • - 쾌속모드 : 차량 내 존재하는 필수 바이러스 제거
    • - 일반모드 : 차량 내 존재하는 바이러스/균 제거
    • - 탈취모드 : 차량 내 담배, 땀 냄새 등 악취 제거
  • ⑤ 연락처 입력 시 작업 종료 후 자동으로 고객에게 소독리포트 전송 (필요 시)
